# 12-Month Review Website - Visual Wireframe & Mockup

## 🎨 Overall Design Concept

**Layout Type:** Single-page application with horizontal tab navigation
**Color Scheme:** IBM Blue (#0f62fe) with professional grays and whites
**Typography:** Clean, modern sans-serif fonts
**Responsive:** Optimized for presentation display (1920x1080)

## 🎨 Design Elements

### Color Palette
```
Primary:   #0f62fe (IBM Blue)
Secondary: #161616 (IBM Gray 100)
Accent:    #42be65 (IBM Green)
Success:   ✅ #24a148
Warning:   ⏳ #f1c21b
Background:#ffffff (White)
Cards:     #f4f4f4 (Light Gray)
Text:      #161616 (Dark Gray)
```

### Interactive Elements
- **Hover Effects:** Cards lift slightly with shadow
- **Click Actions:** Smooth transitions, modal popups for evidence
- **Progress Bars:** Animated fill on page load
- **Tab Navigation:** Active tab highlighted with underline
- **Buttons:** Rounded corners, hover color change

### Typography
```
Headings:  IBM Plex Sans, Bold, 32px
Subheads:  IBM Plex Sans, Semibold, 24px
Body:      IBM Plex Sans, Regular, 16px
Captions:  IBM Plex Sans, Regular, 14px
```

---

## 📱 Responsive Behavior

- **Desktop (1920x1080):** Full layout as shown
- **Tablet (768px):** Stacked cards, collapsible sections
- **Mobile (375px):** Single column, hamburger menu for tabs

---

## ⚡ Interactive Features

1. **Presentation Timer:** Counts up from 00:00, turns yellow at 15min, red at 20min
2. **Progress Tracker:** Shows which sections you've visited
3. **Modal Popups:** Click "View Evidence" to see full-screen images/documents
4. **Smooth Scrolling:** Animated transitions between sections
5. **Keyboard Navigation:** Arrow keys to move between tabs
6. **Print Mode:** Clean, printer-friendly version (Ctrl+P)
7. **Speaker Notes:** Hidden panel with notes for each section (toggle with 'N' key)

---

## 🎯 Key Messaging Focus

**What you want them to remember:**
1. Significant impact and measurable outcomes
2. Continuous learning and skill development
3. Ability to overcome challenges and grow
4. Proactive approach to feedback
5. Well-rounded professional development
6. Clear future direction influenced by IBM

---

## 📝 Content Guidelines

- **Be Specific:** Use numbers, metrics, and concrete examples
- **Show Impact:** Focus on outcomes, not just activities
- **Tell Stories:** Use the STAR method (Situation, Task, Action, Result)
- **Visual Evidence:** Include screenshots, charts, and documents
- **Professional Tone:** Confident but not boastful
- **Time Management:** Ensure content fits 15-20 minute presentation
